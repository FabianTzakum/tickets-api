# Tickets API

API profesional construida con ASP.NET Core para gestionar tickets de soporte técnico.

Este proyecto está diseñado como portafolio backend para demostrar habilidades reales en .NET, arquitectura limpia, APIs REST, SQL Server, Entity Framework Core, autenticación JWT, roles, documentación con Swagger y buenas prácticas.

## Stack

- ASP.NET Core
- .NET 10
- C#
- SQL Server
- Entity Framework Core
- Swagger / OpenAPI
- Visual Studio 2026

## Módulos planeados

- Usuarios
- Roles
- Clientes
- Tickets
- Comentarios
- Estados de ticket
- Prioridades
- Dashboard
- Auditoría básica

## Ejecución en Visual Studio 2026

1. Abrir `tickets-api.sln`.
2. Clic derecho en `Tickets.Api`.
3. Seleccionar `Set as Startup Project`.
4. Elegir perfil `https`.
5. Ejecutar con `Ctrl + F5`.

Swagger debe abrir en:

```txt
https://localhost:7204/swagger
